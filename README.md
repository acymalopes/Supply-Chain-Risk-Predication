# Supply Chain Risk Prediction

🚀 Supply Chain Risk Prediction System

This project implements a machine learning-based supply chain risk prediction system using advanced algorithms to identify and predict potential risks in supply chain operations.

## Setup Instructions

### 1. Create a Virtual Environment

```bash
python -m venv venv
```

### 2. Activate the Virtual Environment

**Windows:**
```bash
venv\Scripts\activate
```

**macOS/Linux:**
```bash
source venv/bin/activate
```

### 3. Install Required Packages

```bash
pip install pandas numpy scikit-learn xgboost imbalanced-learn openpyxl
```

## Running the Script

To run the supply chain risk prediction script:

```bash
python supply_chain_risk_prediction.py
```

## Dependencies

- **pandas**: Data manipulation and analysis
- **numpy**: Numerical computing
- **scikit-learn**: Machine learning algorithms
- **xgboost**: Gradient boosting framework
- **imbalanced-learn**: Handling imbalanced datasets
- **openpyxl**: Working with Excel files

## Project Structure

- `supply_chain_risk_prediction.py` - Main prediction script
- `supply_chain_data.xlsx` - Input data file
- `supply_chain_risk_results.xlsx` - Output results file
